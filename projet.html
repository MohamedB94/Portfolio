Blackjack

Ce projet est le jeu du blackjack. Le blackjack est un jeu de cartes où le but est d'obtenir une main dont la valeur est aussi proche que possible de 21, sans dépasser ce nombre. Chaque carte a une valeur : les cartes numérotées valent leur valeur faciale, les figures (roi, dame, valet) valent 10, et l'as peut valoir 1 ou 11. Les joueurs reçoivent deux cartes au début et peuvent choisir de "tirer" des cartes supplémentaires pour améliorer leur main ou de "rester" avec leur main actuelle. Le joueur gagne s'il a une main plus proche de 21 que celle du croupier, sans dépasser 21.

Pac-Man

Ce projet est le jeu appelé Pac-Man. Le but du jeu est de contrôler Pac-Man, un personnage jaune en forme de cercle, à travers un labyrinthe. Pac-Man doit manger tous les pac-dots (petits points) dans le labyrinthe tout en évitant les fantômes. Si Pac-Man mange un power pellet (gros point), il peut temporairement manger les fantômes pour marquer des points supplémentaires. Le jeu se termine lorsque Pac-Man a mangé tous les pac-dots ou s'il est attrapé par un fantôme.

Floppy-Bird

Ce projet est le jeu Flappy Bird. Le but du jeu est de contrôler un oiseau qui vole en cliquant sur l'écran pour le faire monter et en relâchant pour le faire descendre. L'objectif est de passer à travers des tuyaux sans les toucher. Chaque passage réussi à travers un tuyau augmente le score du joueur.

Tetris

Ce projet est le jeu Tetris. Le but du jeu est d'empiler des blocs de différentes formes qui tombent du haut de l'écran. Le joueur peut déplacer, faire pivoter et positionner les blocs pour former des lignes complètes. Lorsqu'une ligne est complétée, elle disparaît, et le joueur marque des points. Le jeu se termine lorsque les blocs s'empilent jusqu'en haut de l'écran.
